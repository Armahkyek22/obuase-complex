# Classcope Parent App

A React Native application for parents to monitor their children's academic progress, designed to match the provided mockups exactly.

## Features

- View assignments and due dates
- Check attendance records
- Monitor academic results
- Receive notifications
- Manage profile settings

## Tech Stack

- React Native
- Expo
- React Navigation
- TypeScript

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. Scan the QR code with Expo Go on your mobile device, or press:
   - `a` to open Android emulator
   - `w` to open web browser

## Project Structure

```
src/
├── components/     # Reusable UI components
├── navigation/     # Navigation setup
├── screens/        # Screen components
├── utils/          # Utility functions
└── assets/         # Images and other assets
```

## Available Scripts

- `npm start` - Start the development server
- `npm run android` - Run on Android emulator
- `npm run ios` - Run on iOS simulator (macOS only)
- `npm run web` - Run in web browser

## Design Implementation

This application has been designed to match the provided mockups exactly:
- Clean, flat UI elements with no unnecessary inner shapes
- Consistent color scheme and typography
- Proper spacing and alignment
- Responsive layout for different screen sizes
